### Prerequisites

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com//srizzling/mygolangproject.git
   ```
2. Install and tidy up go modules
   ```sh
   make vendor
   ```
3. Build a binary
   ```sh
   make build
   ```
